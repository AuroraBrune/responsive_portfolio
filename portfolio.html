This Eat Da Burger app was fun. We applied some backend skills on this assignment, and learned how to store data in a database using MySQL. The user can submit a burger of their choice, and click on devour it button to consume the burger. The burger is saved in the database even after being devoured. The burger is moved to the right side of the screen upon devouring.

Link: Burger Link: GitHub
